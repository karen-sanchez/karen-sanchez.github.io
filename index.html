What I do.

My interest lies mainly on user experience, user interface, and front-end development. I enjoy building based on research and developing based on good design. I truly enjoy what I do, because it allows me to explore creativity as well as logic.

How.

Web development with design in mind.

Web development is only part of it, the truth is that every great website starts with a great idea. Eventually that idea reaches the point where it needs UX, UI, and development — this is where I like to jump in. I believe that together, these three diciplines can turn great ideas into great websites.

User Experience

Even a small sample of test users can give us insights that we never considered before.

User Interface

Proximity, alignment, repetition, contrast, colors and typeface. All of these must come together to make a nice website.

Front-end Development

A combination of HTML, CSS, and JS are what will make the website come alive and ready to interact with users.
